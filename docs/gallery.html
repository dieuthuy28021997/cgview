Use the links on this page to view sample maps. See creating CGView maps for information on downloading and using CGView. Note that CGView can be used to display any type of feature information, from Gene Ontology classifications to sequence similarity scores.

An examples of a navigable map created by CGView:
  • Example 1 - Roseobacter denitrificans genome. Format: PNG and SVG.
Examples of static images created by CGView:
  • Example 2 (337 kilobytes) - Halocaridina rubra Holthuis mitochondrial genome (map courtesy of Scott Santos, Auburn University). Zoom: unzoomed. Format: PNG.
  • Example 3 (955 kilobytes) - Large map showing genes, COGs, GC content and GC skew. Zoom: unzoomed. Format: PNG.
  • Example 4 (1,424 kilobytes) - Large map of Chlamydia trachomatis D/UW-3/CX. Zoom: unzoomed. Format: PNG.
  • Example 5 (767 kilobytes) - Large map of Chlamydia trachomatis D/UW-3/CX. Zoom: 10X zoom on base 100,000. Format: PNG.
  • Example 6 (384 kilobytes) - Salmonella enterica plasmid pHCM2. Zoom: unzoomed. Format: PNG.
  • Example 7 (2 megabytes) - Escherichia coli. Zoom: 6X zoom on base 4000000. Format: SVG.
  • Example 8 (536 kilobytes) - pBlueScript SK-. Zoom: unzoomed. Format: SVG.
  • Example 9 (4,545 kilobytes) - Large map of Streptococcus pneumoniae R6. Zoom: unzoomed. Format: PNG.
  • Example 10 (13 megabytes) - Reclinomonas americana mitochondrion complete genome. Zoom: unzoomed. Format: SVG.
  • Example 11 (601 kilobytes) - Temperate bacteriophage Aaphi23 of Actinobacillus actinomycetemcomitans. Zoom: unzoomed. Format: PNG.
  • Example 12 (814 kilobytes) - Map showing BLAST comparison results between Aster yellows witches'-broom plasmid pAYWB-III and plasmids pAYWB-I, pAYWB-II, and pAYWB-IV. Zoom: unzoomed. Format: PNG.
  • Example 13 (7.3 megabytes) - Map showing BLAST comparison results between Reclinomonas americana mitochondrion and the E. coli genome, Arabidopsis mitochondrion, and human mitochondrion. Zoom: unzoomed. Format: PNG.
  • Example 14 (18 megabytes) - Large map showing BLAST comparison results for translated CDS features from Methanocaldococcus jannaschii DSM 2661 vs translated CDS features from 100 bacterial genomes. Zoom: unzoomed. Format: PNG.
  • Example 15 (28 megabytes) - Montage showing BLAST comparisons of translated CDS features from various E. coli plasmids. COGs are also shown, in the first and fourth tracks (counting from outside of circles). Zoom: unzoomed. Format: PNG.
  • Example 16 (8 megabytes) - Map comparing E. coli CFT073 CDS translations to CDS translations from all E. coli chromosomes in RefSeq. COG categories and sequence features are shown in the outermost four rings.
  • Example 17 (70 megabytes) - Same as the previous example but drawn at a larger size (12000 x 12000, or 144 megapixels) and with gene labels. To view this image right-click the link and choose Save Link As (or some other similar option) and then open the image in a dedicated graphics program.
